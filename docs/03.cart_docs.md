# 🛒 장바구니 페이지 프로젝트 (Cart Page Project)

## 📘 프로젝트 개요
이 프로젝트는 패션 쇼핑몰의 **결제 프로세스 개선**을 목표로 한 장바구니 페이지 개발입니다.  
기존에는 상품 선택 후 바로 결제 페이지로 이동하는 구조였으나,  
사용자가 결제 전 **상품 정보와 금액을 명확히 확인할 수 있는 단계**를 추가하여  
보다 직관적이고 신뢰성 있는 결제 환경을 제공합니다.

---

## 🎯 프로젝트 목표 및 주요 기능
- **사용자 친화적 장바구니 페이지 구현**
  - 상품 가격, 수량, 배송비, 총 금액 등을 한눈에 확인 가능
- **배송 정책 자동 반영**
  - 10만원 이상 구매 시 배송비(₩3,000) 자동 무료 처리
- **결제 모듈 연동**
  - ‘결제하기’ 버튼 클릭 시 결제 페이지로 매끄럽게 연결
- **반응형 디자인**
  - 모바일(375px 기준)과 데스크톱 모두에서 최적화된 화면 제공

## ⚙️ 설치 및 실행 방법

로컬 환경에서 프로젝트를 실행하려면 다음 명령어를 입력하세요.

```bash
# 저장소 클론
git clone https://github.com/your-repo/cart-page.git
cd cart-page

# 패키지 설치
npm install

# 개발 서버 실행
npm start
