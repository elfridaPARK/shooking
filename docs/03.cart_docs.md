# 🛒 장바구니 프로젝트 (Cart Page Project)

## 📘 프로젝트 개요

이 프로젝트는 **패션 쇼핑몰 웹서비스**의 결제 프로세스 개선을 목표로 한 장바구니 페이지 개발 프로젝트입니다.  
기존에는 상품 선택 후 바로 결제 단계로 이동하는 구조였으나,  
사용자가 결제 전 **상품 정보, 금액, 배송비 등을 명확히 확인할 수 있도록 하는 UI**를 추가하여  
보다 안정적이고 투명한 구매 경험을 제공하는 것이 주요 목표입니다.

### 🎯 주요 기능
- **상품 정보 표시:** 상품 이미지, 브랜드명, 가격, 수량 등을 한눈에 확인  
- **금액 계산:** 상품 수량 조정 시 실시간으로 총 금액 및 배송비 자동 계산  
- **배송 정책 반영:** 10만원 이상 구매 시 배송비(₩3,000) 무료 처리  
- **결제 연동:** ‘결제하기’ 버튼 클릭 시 결제 모듈과 연동  
- **반응형 디자인:** 모바일(375px 기준) 및 데스크톱 모두 대응  

### 👥 참여한 팀원
- **기획 / PM:** Ooo9
- **프론트엔드 개발:** FE 개발팀  
- **디자인:** 디자인팀 (Figma UI 시안 제작 및 검수)

## 🧩 설치 및 실행 방법

로컬 환경에서 프로젝트를 실행하려면 아래 명령어를 입력하세요.

```bash
# 저장소 클론
git clone https://github.com/your-repo/cart-page.git
cd cart-page

# 패키지 설치
npm install

# 개발 서버 실행
npm start
