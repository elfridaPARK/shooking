# 🛍️ 쇼핑몰 연동

## 프로젝트 개요

* React 기반으로 제품 조회, 장바구니, 결제 기능을 구현한 쇼핑몰 연동 프로젝트입니다.
* 사용자는 메인 페이지에서 제품을 확인하고, 상세 페이지에서 제품 정보를 확인한 뒤 장바구니 및 결제 기능을 이용할 수 있습니다.
* **참여한 팀원**

  Ooo9

---

## 개발 환경

* **Framework:** React 17.0
* **Language:** JavaScript ES6
* **State Management:** Context API / Redux / Recoil (선택하여 기재)
* **Styling:** CSS Module / Styled-components
* **Build Tool:** CRA 또는 Vite
* **Package Manager:** npm

## 주요 함수와 기능

### ● 제품 조회 기능

* 메인 페이지에서 전체 제품 목록을 렌더링
* 제품 클릭 시 상세 페이지 이동

### ● 장바구니 기능

* 제품을 장바구니에 추가
* 헤더 장바구니 카운트 증가
* 제품 수량 증가/감소
* 페이지 이동 후에도 수량 유지

### ● 결제 기능

* 장바구니 페이지에서 결제하기 클릭 시 결제 완료 페이지 이동
* 결제 이후 장바구니 초기화 여부 선택 가능

## 테스트 방법

### 명령어 기반 테스트

```bash
npm test
```

### 기능 테스트 체크리스트

1. 제품 카드 클릭 → 제품 상세 페이지로 이동되는지 확인
2. 제품을 장바구니에 담으면 헤더 장바구니 숫자가 증가하는지 확인
3. 결제하기 클릭 시 결제 완료 페이지가 정상 표시되는지 확인
4. 장바구니 수량 조절 후 다른 페이지로 이동해도 변경 내용이 유지되는지 확인

## 유의사항 및 알려진 이슈

* 새로고침 시 장바구니 상태가 초기화될 수 있음
* 모바일 환경에서 레이아웃 미세 조정 필요
* API 연동 시 CORS 설정 필요
* 결제 기능은 Mock API 기준으로 동작하므로 실제 결제 시스템과는 다를 수 있음
