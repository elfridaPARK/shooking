🛍️ 쇼핑몰 연동 리뷰
고객사 요구사항

이메일에 기재된 기본 요구사항 외에도 피그마(Figma) 시안에 포함된 UI/UX 세부 요소를 기준으로 기능을 검증하였습니다.

📌 주요 요구사항 정리

메인 페이지에서 상품 목록이 정상적으로 노출될 것

상품 클릭 시 상품 상세 페이지로 이동할 것

장바구니 담기 기능이 정상동작하고 헤더 아이콘의 카운트가 증가할 것

상세 페이지 및 장바구니 페이지에서 상품 수량 조절 기능 제공

수량 변경 후 다른 페이지로 이동해도 값이 유지될 것

결제하기 버튼 클릭 시 결제 완료 페이지로 이동할 것

전체 페이지 이동 간 Header/Footer는 공통 레이아웃으로 노출될 것

피그마 시안 기준으로 버튼 스타일, 폰트, 컬러, 간격 등 UI 요소를 적용할 것

장바구니 개수 표시는 최대 99+ 형태로 표현 (피그마 기준 적용 가능 시)

로딩 상태 및 오류 처리(UI) 반영

요구사항에 따른 기능 동작 확인

고객사 요구사항 + 피그마 시안에서 확인한 세부 기능을 기준으로 실제 동작 여부를 점검하였습니다.

✔️ 1. 제품 상세 페이지 이동 여부

메인 페이지에서 제품 카드 클릭 시 /product/:id 경로로 정상 이동

피그마 시안 기준 상세 페이지 레이아웃과 구성요소(이미지 / 이름 / 설명 / 가격) 정상 표시

뒤로가기 및 상단 로고 클릭 시 메인 페이지로 정상 복귀

✔️ 2. 장바구니 기능 정상작동 여부

메인 페이지 및 상세 페이지에서 장바구니 담기 버튼 클릭 시 헤더 장바구니 아이콘 숫자 증가

동일 제품을 여러 번 담을 경우 수량이 누적되어 반영

장바구니 페이지에서 제품 수량 증가/감소 버튼 정상 작동

수량이 1 이하로 내려가지 않도록 예외 처리 확인

피그마 기준 버튼 스타일, 아이콘 위치, 색상 등의 UI 요소 정상 반영

✔️ 3. 결제 기능 동작 여부

장바구니 페이지 → 결제하기 버튼 클릭 시 /payment/complete 페이지 정상 이동

결제 완료 페이지에서 주문 완료 문구, 확인 버튼, 주문 요약 정보 시안대로 표시

결제 완료 후 장바구니 초기화(옵션 적용 여부 확인 필요)

✔️ 4. 상태 유지 기능 확인

장바구니/상세 페이지에서 수량 변경 후 다른 페이지로 이동

다시 장바구니 페이지 방문 시 변경된 수량 정상 유지

LocalStorage 또는 전역 상태관리 로직 문제 없음

✔️ 5. UI / UX 검증 (피그마 반영 여부)

전체 폰트 스타일, 사이즈, 컴포넌트 간격, 컬러 피그마와 일치

버튼 hover, active 상태 반영

모바일/태블릿 대응 여부 확인 (프로젝트 범위에 따라 다름)

이미지 비율 유지 및 깨짐 현상 없음

✔️ 6. 기타 세부 기능 확인

헤더 장바구니 아이콘 클릭 시 장바구니 페이지로 정상 이동

잘못된 상품 ID 접근 시 오류 처리 페이지 또는 리다이렉션 여부 확인

빈 장바구니 페이지 시 시안에 맞는 빈 상태 화면 제공

페이지 이동 시 스크롤 정상 초기화
